# Contribution Guidelines

First off, thank you for considering contributing to Awesome Computer Architecture! 

Please ensure your pull request adheres to the following guidelines:

- Search previous suggestions before making a new one, as yours may be a duplicate.
- Make sure the link is functional (no 404s).
- Ensure the item is highly relevant to Computer Architecture.
- Keep descriptions short, clear, and objective.
- Add the item to the correct category in alphabetical order.

**Format:**
`- [Name](link) - Description.`
(Description must start with a capital letter and end with a period).

Thank you for your suggestions!
